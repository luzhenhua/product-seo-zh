## Module 1：H1 + 首屏

**执行前必做**：
1. 拿到阶段 0.7 的「竞品 brief」。本模块的 Hero 段是用户阅读产品页的第一印象，**最容易模板化**，必须吃 brief 做差异化。
2. 查阶段 0.5 的「Hero UI 容量」变量：未指定 → **默认走短版本**（多数出口品牌站的真实约束，见 [reference/user-context.md](../reference/user-context.md)）；用户明确说"长版本" → 走长版本。

═══════════════════════════
SHARED RULES（短/长版本通用）
═══════════════════════════

**H1 规则**（短长通用）：
- Get to the point. No "Welcome to..." preamble.
- Include core keyword + value modifier
- 8-15 words
- Format reference: "Rod Mill Manufacturer & Supplier — Efficient Grinding Solutions for Mining"

**首段品牌词硬限制**（短长通用，绝对不能违反）：
- 整个 Hero Paragraph **品牌词总出现次数 ≤ 1**
- 计数对象：品牌简称（如 "Zhongyi"）+ 全称 + "Our [产品]" + "[品牌]'s [产品]"
- 推荐 0 次或 1 次。1 次时只能放最后一句（解决方案句）
- 自查：写完数一遍。≥ 2 立刻重写

**第一句硬限制**（短长通用）：
- ❌ 禁止以品牌词开头（"[Brand] is a..." / "[Brand] provides..." / "Our [产品]..."）
- ✅ 必须以产品定义开头（"An impact crusher is..." / "A rod mill is..." / "A diesel jaw crusher is..."）

**FORBIDDEN openings**（短长通用）：
In today's world / With the development of / In recent years / It is worth noting / When it comes to / One of the key

**FORBIDDEN words**（短长通用）：
comprehensive / paramount / holistic / delve into / robust / cutting-edge / game-changer / revolutionize / unlock / leverage / enhance / optimize / seamless / innovative

**CTA**（短长通用）：
Text: "Request a Quote" OR "Get a Quote Now" OR "Request a Solution"

- 阶段 0.5 未指定 Hero UI 容量 → **默认短版本**
- 阶段 0.5 用户明确说"长版本" / "我们网站 Hero 装得下" → 走长版本

- 长版本字数预算：H1（8-15 词）+ paragraph（70-105 词）+ 4 bullets
- 首屏 paragraph 用品牌简称（如 Zhongyi），不要用全称
- 参数值要合理范围，不能太整（如不要写 100, 200, 300, 400 这种）

- ⚠️ **结构自查（短版本）**：Sentence 1 是否产品定义（不含品牌词）？Sentence 2 是否能力/参数？任一项不符立刻重写
- ⚠️ **结构自查（长版本）**：4 句结构是否完整（定义/特点/功能/解决方案）？任一项不符立刻重写
