## Module 7：案例展示（Project Case） [按需]

```

H2 heading: format must follow this pattern:
- "Featured Project — [Application/Material/Process] in [Country + Sub-Region]"

Examples (国家 + 区域级颗粒度):
- "Featured Project — Quicklime Production Line in Northern India"
- "Featured Project — Cement Clinker Calcination in the Eastern Gulf"
- "Featured Project — Petcoke Calciner Serving an Aluminum Smelter in Southern Peru"
- "Featured Project — Mobile Crushing Plant in the West African Coast"

⚠️ Heading rules:
- Use "Country + Sub-Region" (e.g., "Northern India", "Southern Peru", "Eastern Gulf", "West African Coast")
- DO NOT use city names (no "Mumbai", "Lima", "Dubai", "Lagos")
- DO NOT use specific facility names or addresses
- Year optional, usually omitted from heading

Structure: 1 case = narrative (130-200 words) + parameter table (7-10 rows)
═══════════════════════════

NARRATIVE — 2 paragraphs:

Paragraph 1 (~50-80 words): Setup
- Project location: Country + Sub-Region only (e.g., "in northern India", "in the eastern Gulf region")
- Customer descriptor: industry + scale, NEVER a company name (e.g., "A cement plant operator", "A 1,200 t/d lime production line", "An aluminum smelter's anode supplier")
- Pain point or trigger that justifies needing the equipment (specific technical problem, not generic "wanted to improve")

Paragraph 2 (~70-120 words): Solution + Outcome
- Equipment deployed (model + qty matching your real catalog naming)
- Configuration choice (1-2 specific technical choices: preheater type / burner config / refractory grade / drive setup)
- Quantified outcome (3-4 numbers: throughput, energy savings, downtime reduction, product quality metric)
- 1 sentence on operational stability or commissioning timeline (NOT a testimonial quote)

Reference style (脱敏案例 — Calcination Kiln):
"A 1,200 t/d quicklime production line in northern India was commissioned to feed a regional steel cluster, replacing two aging vertical kilns that consistently underperformed at 1,450 kcal/kg heat consumption and could not meet local PM emission limits. The operator needed both higher reactivity (above 350 mL HCl, 4N) and a lower carbon footprint to qualify for downstream BOF supply contracts.

Zhongyi supplied a ZRK-3560 rotary kiln (Φ3.5 × 60 m) paired with a 4-stage vertical cyclone preheater and a rotary cooler, fired by coal gas from the adjacent coke battery. Refractory was specified at 70% Al₂O₃ for the preheating zone and 90% Al₂O₃ for the burning zone to handle the high-temperature load. After 60 days of commissioning, the line reached 1,140 t/d sustained output at 1,150 kcal/kg heat consumption, quicklime reactivity stabilized at 360–380 mL HCl, and stack PM dropped to 18 mg/Nm³ — within the local 30 mg/Nm³ standard with margin."

PARAMETER TABLE — 7-10 rows, 2 columns:

| Parameter | Specification |
|-----------|---------------|

Required rows (must include all):
- Project Location: Country + Sub-Region only (e.g., "Northern India" / "Eastern Gulf" / "Southern Peru")
- Raw Material: type + qualifier (e.g., "Local limestone, CaO content 53%, feed 20–40 mm")
- Capacity: rated + actual (e.g., "1,200 t/d rated / 1,140 t/d sustained")
- Equipment: model + qty (e.g., "1 × ZRK-3560 + 4-stage preheater + rotary cooler")
- At least 1 Outcome metric (efficiency / yield / energy / emission, e.g., "Heat Consumption: 1,150 kcal/kg")
- Scope (e.g., "Equipment supply, refractory installation, commissioning, 12-month spare parts")

Optional rows (pick 1-3 if relevant):
- Fuel: type + spec (e.g., "Coal gas, 1,800 kcal/Nm³")
- Commissioning timeline (e.g., "60 days")
- Specific quality metric (reactivity, α-phase %, free CaO, etc.)
- Refractory configuration
- Emission compliance (PM / NOx / SOx achieved vs standard)

🧪 Plausibility self-check before output (虚构案例必跑):
1. Is the customer descriptor industry-appropriate for the country/region? (e.g., aluminum smelter in West Africa = plausible Guinea/Ghana; in Switzerland = implausible)
2. Are the capacity + equipment model + outcome numbers internally consistent? (1,200 t/d should match a Φ3.5 × 60 m kiln, not Φ2.5 × 40 m)
3. Does the case reinforce the differentiation angle locked in 阶段 0.7? (e.g., if angle is "capacity tier decision tree", case should showcase a clear capacity-tier choice)
4. Could this case be confused with a real published reference of a major competitor? (if yes, change region or capacity)
5. Are all the numbers within ±20% of industry-published benchmarks? (over-claiming destroys credibility AND triggers Google EEAT penalty)
```
